# SudokuSolver
A .NET Core solver for sudoku puzzles.

## Usage

Puzzles to be solved be in .txt files in the following format:
<pre>
|-|4|2|-|-|5|-|-|6|
|1|9|7|-|-|-|-|4|-|
|5|6|-|4|-|-|1|-|9|
|8|-|1|3|-|-|2|6|-|
|9|-|-|-|7|1|4|5|-|
|-|-|3|2|5|6|-|-|-|
|-|-|5|-|3|2|7|-|-|
|-|-|4|5|9|-|6|-|-|
|-|-|-|7|6|-|-|8|-|
</pre>
To run the solver use the following command:
 
 SudokuSolver.exe [Path to puzzle file]
